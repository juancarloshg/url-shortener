https://aqueous-anchorage-96750.herokuapp.com/new/your-URL will return you a json object with your URL and a new shortened URL.

Now, you can visit your shortened URL and it will redirect you to the URL you passed before.

For example, https://aqueous-anchorage-96750.herokuapp.com/new/https://www.google.com gave me https://aqueous-anchorage-96750.herokuapp.com/0, if you visit that it will redirect to https://www.google.com
